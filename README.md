# Análisis-de-Fotos-Hemisféricas-para-Zonas-Verdes
Es una fotografía tomada con lente "ojo de pez" (180°) desde el suelo hacia arriba, que captura la cobertura vegetal. Se usa para estudiar la estructura del dosel forestal, parques, y áreas verdes urbanas


¿Qué hace este código?
Análisis por orientación:
Norte: Muestra exposición al sol

Sur: Menos exposición directa

Este/Oeste: Exposición matutina/vespertina

Análisis por ángulo cenital:
0-18° (cenit): Cobertura superior directa

72-90° (horizonte): Vista lateral, importante para sombreado

Índices calculados:
IVC: Índice de Cobertura Vegetal

LAI: Índice de Área Foliar

IPL: Penetración lumínica

Clasificación de calidad:
>80%: Excelente (bosque maduro)

60-80%: Buena (parque arbolado)

40-60%: Regular (áreas semiurbanas)

20-40%: Mala (zonas con poca vegetación)

<20%: Muy mala (zonas urbanas densas)

Aplicaciones prácticas:
  Planificación urbana: Dónde plantar más árboles

  Evaluación de parques: Calidad de zonas verdes

Estudios climáticos: Efecto de isla de calor

Bienestar urbano: Acceso a espacios verdes

Conservación: Monitoreo de cobertura vegetal

Este análisis es fundamental para ciudades sostenibles, ya que las zonas verdes:

Reducen temperatura urbana

Mejoran calidad del aire

Incrementan bienestar psicológico

Reducen contaminación acústica

Proveen hábitat para biodiversidad
